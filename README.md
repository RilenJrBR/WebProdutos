
# Desafio_Produtos
===============================

## Descrição

- Uma API com endpoints para cadastrar, atualizar e apagar os produtos (CRUD);

## Funcionalidades

- Cadastrar produtos
- Obter produto
- Listar todos os produtos
- Atualizar produto
- Deletar produto

## Tecnologias Utilizadas

- Backend: C#
- ASP.NET Web Forms
- .NET CORE
- .NET6
- PostgreSQL.
